Features

Email/Password Auth

Registration and login with bcrypt password hashing (14 salt rounds) and server-side Zod schema validation.

Multi-Factor Authentication

Email-based 6-digit OTP codes with bcrypt-hashed storage, 60-second resend cooldown, and automatic lockout after 10 failed attempts.

Social Sign-On

Google and Microsoft OAuth via OpenID Connect with PKCE code challenge for security. Supports account linking.

Session Management

JWT session tokens (90-day TTL) with ES256 signing, dual-key rotation for access tokens, and MongoDB TTL-based cleanup.

Account Linking

Connect multiple SSO providers to a single account. Unlink providers without losing access.

API Documentation

Full OpenAPI 3.0 specification with Swagger UI available in development mode.
